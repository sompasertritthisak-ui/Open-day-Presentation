Automated Communications

Branded Email Automation System

🔔 New Registration
→ Admissions Team
Branded HTML notification with full parent details, campus, children count, translator flag, and ACTION REQUIRED banner. Sent to admissions@pbis.edu.la.
✅ Tour Approved
→ Parent
Confirmed visit: Date, Time, Campus, Location with Google Maps button. Green header. PBIS logo top-left. "Arrive by 8:45 AM" instruction.
❌ Tour Declined
→ Parent
Professional decline with red header. Warm tone maintained. Encourages re-engagement. Admissions contact details included.
📅 Rescheduled
→ Parent
Purple header with proposed new date/time highlighted. Parent prompted to confirm. Direct contact details for questions.
💛 We Missed You
→ Parent (No-Show)
Warm follow-up for Open Day no-shows. Offers private campus tour. Optional personalised note added by admissions team.

Architecture & Security

Technology Stack & Security Features

🛠 Technology Stack
▸Single-file HTML — no frameworks, no dependencies
▸Vanilla JavaScript — maximum browser compatibility
▸Google Apps Script — serverless backend
▸Google Sheets API — real-time database
▸Gmail API — branded email delivery
▸GitHub Pages + Custom Domain (openday.pbis.edu.la)
▸Google Translate — 31 language support
▸Panoee / Kuula — 360° virtual tour embeds
▸Zero monthly cost — built on free-tier services
🔒 Security Features
▸SHA-256 hashed password — never stored in plain text
▸5-attempt lockout with 60-second cooldown
▸2-hour session expiry with automatic logout
▸XSS protection — all user inputs sanitised
▸Rate limiting on form submissions (max 3/hour)
▸CORS-safe API calls via URL-encoded POST
▸Pending mutations queue — data integrity protected
▸Safari-compatible — delegated event system
▸Data stored only in Google Sheets & Gmail — PBIS-owned

End-to-End Journey

From Parent Discovery to Enrolled Student

1
🌐 Parent Discovers
Visits openday.pbis.edu.la via social media, WhatsApp, Google Search or referral
2
👀 Explores Page
Views campus photos, 360° virtual tour, promotions, and Open Day details
3
📝 Registers
Fills registration form. Auto-email sent to parent. Admissions team notified instantly.
4
⚙️ Team Reviews
Admissions team logs into dashboard. Approves, Declines, or Reschedules.
5
✅ Parent Confirmed
Branded confirmation email sent. Visit details, map link, and contact info.
6
🏫 Open Day Visit
Family attends. Enrollment status tracked in Pipeline. Overdue flagged if absent.
7
💌 Follow-Up
No-shows receive personalised "We Missed You" email. Private tour offered.
8
🎓 Enrolled
Family enrols. Status updated to Enrolled across all Google Sheets tabs.
